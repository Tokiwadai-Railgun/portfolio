# Présentation
Le TypeScript est une surcouche de ``JavaScript`` rajoutant des types. Très utile pour éviter les erreurs communes de types rencontrés en JS surtout quand on traite avec des API je l'utilise désormais pour remplacer le JavaScript dans la pluspart de mes projets légers / Projets web. 
# Utilisation 
Comme dit précédement le typescript prend sa place dans tous mes projets léger / projets orientés développement. Par exemple se site est réalisé avec ``svelte`` et ``typescript``
